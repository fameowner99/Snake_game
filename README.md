# Snake_game
Simply snake game using dynamic libraries for GUI and sounds

Works only on MACOS

How to install and run:
1) download
2) sh download.sh (script which download sdl and sfml libraries)
3) make
4) ./nibbler X Y (X and Y are numbers)
